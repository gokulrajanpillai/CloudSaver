# Paid Convenience Builds

CloudSaver's core remains open source. Paid convenience builds can fund maintenance without
closing the scanner, dashboard, duplicate review, quarantine, or local report features.

## What Stays Free

- Source code
- CLI
- Local web app
- Scan engine
- Duplicate verification
- Quarantine and restore primitives
- JSON and CSV exports
- GitHub release source artifacts

## What Can Be Paid

- Signed macOS and Windows builds
- Store-distributed builds
- Automatic update convenience
- Priority support
- Commercial support terms
- Business report templates
- Implementation help for small teams

## Suggested Channels

| Channel | Price | Notes |
| --- | ---: | --- |
| GitHub Releases | Free | Source, checksums, and preview artifacts |
| GitHub Sponsors | Monthly | Supports open-source maintenance |
| Gumroad or Ko-fi | Pay what you want | Simple signed-build distribution |
| Mac App Store | $4.99-$9.99 | Convenience, trust, and updates |
| Microsoft Store | $4.99-$9.99 | Convenience, trust, and updates |
| Commercial Support | $299+/year | Support for professional users and small teams |

## Rules

- Do not hide basic safety features behind a paywall.
- Do not add ads, hidden telemetry, or data sales.
- Do not require an account for local scanning.
- Make paid convenience clearly about packaging, trust, updates, and support.
