# Screenshots

CloudSaver needs product screenshots before a broad public launch.

Recommended screenshots:

- First-run folder selection.
- Scan progress.
- Dashboard summary.
- Storage treemap.
- Duplicate review.
- Quarantine/review flow.
- Recent scan history.

Store screenshots under `docs/assets/` and reference them from `README.md`.

Suggested filenames:

- `docs/assets/dashboard.png`
- `docs/assets/storage-map.png`
- `docs/assets/duplicate-review.png`
- `docs/assets/quarantine.png`
