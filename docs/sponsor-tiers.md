# Sponsor Tiers

Use these tiers when configuring GitHub Sponsors.

## $3/month - Supporter

For users who want to keep CloudSaver maintained.

Benefits:

- Sponsor updates
- Name listed in periodic thank-you notes when practical

## $10/month - Power User

For users who rely on CloudSaver regularly.

Benefits:

- Roadmap voting
- Sponsor updates
- Early feedback threads

## $25/month - Professional

For professional users and consultants.

Benefits:

- Priority issue triage where feasible
- Early release notes
- Roadmap voting

## $100/month - Small Business

For small teams using CloudSaver for shared-drive cleanup.

Benefits:

- Name or logo in sponsor acknowledgements, if desired
- Priority issue triage where feasible
- Business-report feedback channel

## $500/month - Project Sponsor

For organizations funding platform maintenance.

Benefits:

- Prominent acknowledgement in release notes and README, if desired
- Input on packaging and platform priorities
- Commercial support conversation

## Sponsor Messaging

CloudSaver should explain that sponsorship funds:

- Signed builds
- Release automation
- Platform packaging
- Safer cleanup workflows
- Dashboard polish
- Documentation and screenshots
