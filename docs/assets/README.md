# Screenshot Assets

Store current CloudSaver preview screenshots in this directory before tagging
`v0.1.0-preview`.

Required files:

- `first-run-scan.png`
- `scan-progress.png`
- `dashboard-summary.png`
- `storage-map.png`
- `duplicate-review.png`
- `review-restore.png`
- `history.png`

Screenshots should use a disposable fixture folder and must not show private paths, file
names, email addresses, license keys, or customer data.
