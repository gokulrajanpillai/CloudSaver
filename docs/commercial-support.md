# Commercial Support

CloudSaver can remain open source while offering paid help for professionals and small
teams that need reliable storage audits.

## Support Packages

| Package | Suggested Price | Scope |
| --- | ---: | --- |
| Individual Pro Support | $299/year | Priority issue triage, release guidance, and setup help |
| Small Team Support | $999/year | Shared-drive audit guidance, report templates, and scheduled check-ins |
| Cleanup Consultation | $99/session | One-off help interpreting a scan and cleanup plan |

## Included Work

- Installation and release artifact guidance.
- Scan setup for local drives, synced cloud folders, external drives, and NAS mounts.
- Report interpretation.
- Safe cleanup planning.
- Business report customization.

## Not Included

- Access to private user files.
- Hosted file storage.
- Secret telemetry.
- Guaranteed data recovery.
- Silent deletion workflows.

## Principles

Commercial support should fund the open-source project while keeping the core product
available to everyone.
